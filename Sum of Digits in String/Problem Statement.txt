Write a Java program that takes a single line string from user including characters and digits.
Calculate the sum of all digits in the string. 
If the sum of digits is equal or less than 15 then print the sum as it is.
If the sum is greater than 15, or if no digits are found, print "No Digit Found".


Sample Input 1: "GTAen9y4"
Sample Output 1: 13

Sample Input 2: "pue8QM9"
Sample Output 2: "No Digit Found"

Sample Input 3: "helloWorld"
Sample Output 3: "No Digit Found"

Write the code in the  class Solution { }.
Use the Scanner class to take user input.
